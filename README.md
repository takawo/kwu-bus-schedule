# 甲南女子大学スクールバス運行ダイヤアプリ（非公式）

甲南女子大学のスクールバス運行状況を確認できるWebアプリケーションです。

## 🌐 公開URL

**<https://takawo.github.io/kwu-bus-schedule/>**

GitHub Pagesで公開されています。

## 機能

- **リアルタイム運行状況表示**: 現在時刻に基づいて次のバスの発車時刻を表示
- **ルート切り替え**: 本山発と大学発の2つのルートを切り替えて表示可能
- **今日の運行ダイヤ**: 当日の全運行ダイヤを一覧表示
- **シャトル運行対応**: シャトル運行時間帯を識別して表示
- **時刻設定機能**: 任意の時刻を設定して運行状況を確認可能

## データ形式について

スケジュールデータはJSON形式で保存されています：

- `bus_schedule_motoyama.json`: 本山スクールバスのりば発の運行ダイヤ
- `bus_schedule_daigaku.json`: 大学発本山スクールバス乗り場着の運行ダイヤ

各JSONファイルには以下の情報が含まれています：

- **定時発車**: 固定時刻に発車する便
- **シャトル運行**: 満車になり次第出発する時間帯

## ファイル構成

```
├── index.html          # メインHTMLファイル
├── app.js              # アプリケーションロジック
├── style.css           # スタイルシート
├── bus_schedule_motoyama.json   # 本山発の運行ダイヤデータ
└── bus_schedule_daigaku.json    # 大学発の運行ダイヤデータ
```

## 使用方法

1. ブラウザで公開URLにアクセス
2. 「本山発」または「大学発」ボタンでルートを切り替え
3. 次のバスの発車時刻と今日の運行ダイヤが自動的に表示されます
4. 必要に応じて「時刻を設定」で任意の時刻を設定して確認可能
